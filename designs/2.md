Design:
```
                           Exploterm 1.0
[..][Sort][Find][Bookmarks][Rename][Preview][VS Code][Terminal][Run]
+-----------+-------+-------------+--------------+-----------------+
|    Name   | Type  |  Extension  |     size     |     Modified    |
+-----------+-------+-------------+--------------+-----------------+
|  Code     |Folder |N/A          |1000          |09/06/21         | 
|  main     |File   |.py          |10            |09/06/21         |
+-----------+-------+-------------+--------------+-----------------+
|File: 1 of 2       Path: C:/Users/Expoterm/Desktop         h=Help | 
```
Functionalities + Shortcuts:
```
• Movement:
  - up arrow
  - down arrow
  - Ctrl+up arrow = 10 files/folders up
• Change directory:
  - left arrow = parent directory
  - right arrow/enter = open file/open directory
  - Ctrl+b = add to bookmarks (Number of bookmarks = 10)
• Window:
  - Ctrl+r = refresh window
  - s = sort files
  - / = find in directory
  - r = rename file/directory
  - Ctrl+shift+n = New directory
  - Ctrl+n = New file
  - Ctrl+v = Open file preview
  - b = go back
  - Ctrl+v = Open directory in VS Code
  - Ctrl+t = Open directory in terminal 
  - Ctrl+r+p = run python script
  - q = quit exploterm
  ```
